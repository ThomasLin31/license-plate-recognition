# 簡易車牌辨識
在沒有足夠汽車車牌數據去訓練的情況下簡易地識別車牌

### 檔案說明 ###
* finalpredict.py: 主要程式，用來預測並顯示結果
* getPlateAndNums.py: 用來辨識車牌並取得車牌數字
* resize.py: 用來將數字大小變為28x28，並黑白轉換，以供訓練
* numtrain.py: 用來訓練數字並生成模型
* newModel.h5: numtrain生成出來的模型
